# Node.js DOCX Paper Generator

A simple Node.js experiment that uses the `docx` library to generate a professionally formatted Microsoft Word document automatically.

## Features

* Generate `.docx` files programmatically
* Custom headings and formatting
* Automatic page numbering
* Tables and structured sections
* Header and footer support

## Technologies Used

* Node.js
* npm
* docx

## Purpose

This project was created as a learning exercise to understand:

* Node.js package management
* Document generation using JavaScript
* Working with third-party libraries
* Automation of document creation

## Run Locally

```bash
npm install
node generate.js
```

The generated document will be saved in the project directory.
